09
远程 API 用 curl / 脚本远程加号、刷新、查询的接口示例

管理后台所有接口都在 http://127.0.0.1:3458/admin/* 下。需要在 Authorization: Bearer <admin-key> 携带启动时的 -admin-key。 所有 mutating 请求（POST/PUT/PATCH/DELETE）还需要 X-Requested-With: XMLHttpRequest 头以满足 CSRF 中间件。

列出账号

curl -H "Authorization: Bearer $KEY" \
     http://127.0.0.1:3458/admin/accounts | jq

添加 Kiro 账号（从已有 token 导入）

curl -X POST \
     -H "Authorization: Bearer $KEY" \
     -H "Content-Type: application/json" \
     -H "X-Requested-With: XMLHttpRequest" \
     -d '{
       "id": "kiro-alice",
       "label": "alice@example.com",
       "priority": 100,
       "proxy_url": "http://1.2.3.4:8080",
       "max_in_flight": 3,
       "kiro_auth_token_raw": {
         "accessToken":  "",
         "refreshToken": "",
         "expiresAt":    "2026-12-01T00:00:00Z",
         "profileArn":   "arn:aws:codewhisperer:us-east-1:...:profile/USER_ID",
         "authMethod":   "Social",
         "region":       "us-east-1"
       }
     }' \
     http://127.0.0.1:3458/admin/accounts

批量导入

# 用 cockpit-tools 导出的 JSON 文件直接喂：
curl -X POST \
     -H "Authorization: Bearer $KEY" \
     -H "Content-Type: application/json" \
     -H "X-Requested-With: XMLHttpRequest" \
     -d "{\"mode\":\"append\",\"accounts\":$(cat accounts.json)}" \
     http://127.0.0.1:3458/admin/accounts/import

更新代理 URL

curl -X PATCH \
     -H "Authorization: Bearer $KEY" \
     -H "Content-Type: application/json" \
     -H "X-Requested-With: XMLHttpRequest" \
     -d '{"proxy_url": "socks5://127.0.0.1:1080"}' \
     http://127.0.0.1:3458/admin/accounts/kiro-alice

强制刷新配额

curl -X POST \
     -H "Authorization: Bearer $KEY" \
     -H "X-Requested-With: XMLHttpRequest" \
     http://127.0.0.1:3458/admin/accounts/kiro-alice/refresh

删除账号

curl -X DELETE \
     -H "Authorization: Bearer $KEY" \
     -H "X-Requested-With: XMLHttpRequest" \
     http://127.0.0.1:3458/admin/accounts/kiro-alice

Python (requests) 一键示例

import json, requests

BASE = "http://127.0.0.1:3458"
KEY  = "your-admin-key"
HEAD = {
    "Authorization":     f"Bearer {KEY}",
    "Content-Type":      "application/json",
    "X-Requested-With":  "XMLHttpRequest",
}

# 批量导入 cockpit-tools 导出的 accounts.json
with open("accounts.json") as f:
    accounts = json.load(f)

r = requests.post(f"{BASE}/admin/accounts/import",
    headers=HEAD,
    json={"mode": "append", "accounts": accounts})
r.raise_for_status()
print(r.json())  # {"added": N, "skipped": M}

提示：admin-key 同时是 web 登录密码，把它写进自动化脚本前请确认环境安全。如果需要更细粒度的 deploy token，告诉我，可以另开 /admin/api-keys-deploy。
